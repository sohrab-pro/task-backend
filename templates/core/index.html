Task Management API

A RESTful API for managing your tasks efficiently

Quick Start

The Task Management API provides endpoints for user authentication and task management. All API endpoints are accessible via HTTP requests.

Authentication Required: Most endpoints require a valid authentication token. Include it in your requests using the Authorization header.

Base URL

http://127.0.0.1:8000/api/

Authentication

POST /api/signup/

Create a new user account

{ "username": "your_username", "email": "your_email@example.com", "password": "your_password" }
POST /api/login/

Login to get authentication token

{ "username": "your_username", "password": "your_password" }

Task Management

GET /api/tasks/

Get all tasks for authenticated user

POST /api/tasks/

Create a new task

{ "title": "Task Title", "description": "Task Description", "status": "pending" }
PUT /api/tasks/{id}/

Update an existing task

DELETE /api/tasks/{id}/

Delete a task

Authentication Header Example

Authorization: Token your_auth_token_here
